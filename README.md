# ValStretch

简体中文 | [English](README_EN.md)

VALORANT 真实拉伸分辨率工具。

本工具用于在游戏启动和退出时自动应用 / 恢复拉伸相关设置，减少手动切换分辨率和反复调整配置的麻烦。

> 本软件完全免费，谨防假冒套壳收费版本。

## 主要功能

- 一键应用 / 恢复真实拉伸分辨率。
- 自动模式：游戏启动后自动应用，游戏退出后自动还原。
- 支持常用 4:3 分辨率预设和自定义目标分辨率。
- 支持 16:9 到 4:3 的鼠标灵敏度换算。
- 支持屏幕通知、托盘运行、开机自启动和最小化到托盘。
- 支持国服 / 国际服配置路径分别保存。

## 下载与运行

1. 从官方发布渠道下载 `valstretch-<version>-windows-x64.zip`。
2. 解压后运行其中的 `valstretch.exe`。
3. 首次启动请先阅读并确认「使用须知」。
4. 按照界面提示选择 VALORANT 安装或配置目录，也可以使用「自动检测」。

发布包会同时提供 `SHA256SUMS.txt`。如需校验文件完整性，可在 PowerShell 中执行：

```powershell
Get-FileHash .\valstretch.exe -Algorithm SHA256
```

## 使用步骤

1. 打开工具后，在「高级设置」里选择当前区服，并填写 VALORANT 安装或配置目录。
2. 点击「自动检测」或「检测配置」，确认工具已找到配置文件。
3. 在主界面选择目标分辨率。
4. 推荐开启「自动拉伸」：之后启动游戏会自动应用，退出游戏会自动恢复。
5. 如需临时使用，也可以点击「手动应用」，结束后点击「恢复」。

## 常见问题

### 仍然有黑边怎么办？

先确认目标分辨率是显示器和显卡支持的常见 4:3 分辨率，例如 `1440x1080`、`1280x960`、`1280x1024`。

如果仍然有黑边，请在显卡控制面板或显示器 OSD 中把缩放方式设为「全屏 / Full」。

### 自动模式没反应怎么办？

确认这几件事：

- 当前区服选择正确。
- 已经检测到配置文件。
- 游戏进程已经真正启动。
- 工具没有被安全软件拦截。

### 关闭工具前需要手动恢复吗？

如果工具处于已应用状态，关闭程序或关闭自动模式时会尝试自动恢复。为了稳妥，手动模式下建议结束游戏后点击一次「恢复」。
